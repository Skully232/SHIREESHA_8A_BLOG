### 📝 Project Overview
This repository contains a web blog entry for **"THE DIGITAL VILLAGE: WEB DESIGN COMPETITION"** held at **Bapu Primary-High School**. 

The competition challenged students from Class 7 and 8 to apply their knowledge of **HTML and CSS** to create a digital narrative about rural roots and heritage. Shireesha's project, "Mandya: The Sakkare Nagara," explores the agricultural heartland of Karnataka. The blog highlights Mandya's vital connection to the Cauvery River, its historical significance as a power pioneer, and its famous local delicacies.

### 💻 Technical Skills Demonstrated
* **HTML5 Semantic Structure:** Organized using clear section headings (`<h2>`) and structured lists (`<ul>`) to separate natural wonders, spiritual highlights, and culinary history.
* **CSS3 Styling:** * **Layout Architecture:** Implementation of a 60% width centered blog container with `margin: auto` for optimized readability.
    * **Visual UI Design:** Use of `border-radius` (10px) on the main content box and (8px) on images to create a modern, high-quality aesthetic.
    * **Bold Color Theory:** A striking contrast using a Black background (`#000000`) and a Light Pink content box (`#FFB6C1`), with red headers (`color: red`) to make the cultural content stand out.
    * **Typography:** Effective use of `<b>` (bold) and `<i>` (italics) to emphasize key locations like **Srirangapatna** and historical facts like *Asia’s first hydroelectric power station*.
* **Content Curation:** Comprehensive research into the region, covering the twin cascades of **Shivanasamudra Falls**, the legendary **Maddur Vada**, and the migratory birds of **Ranganathittu**.
